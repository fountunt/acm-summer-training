# 欧拉筛（线性筛）$O(n)$

## 概念

线性筛每个合数只被其**最小质因子**筛掉一次，保证 $O(n)$ 复杂度，同时可以方便地计算积性函数。

---

## 代码

```cpp
#include <bits/stdc++.h>
using namespace std;

/**
 * @brief 欧拉筛（线性筛），筛出 [2, n] 所有质数
 *
 * @param n  上界
 * @return   {primes, minp}
 *   - primes: 质数列表
 *   - minp:   minp[i] = i 的最小质因子（合数）或 0（质数）
 */
pair<vector<int>, vector<int>> linear_sieve(int n) {
    vector<int> primes, minp(n + 1, 0);
    for (int i = 2; i <= n; ++i) {
        if (!minp[i]) {             // i 是质数
            minp[i] = i;
            primes.push_back(i);
        }
        for (int p : primes) {
            if (p > minp[i] || i * p > n) break;
            minp[i * p] = p;        // p 是 i*p 的最小质因子
        }
    }
    return {primes, minp};
}
```

---

## 使用示例

```cpp
int main() {
    ios::sync_with_stdio(false); cin.tie(nullptr);

    auto [primes, minp] = linear_sieve(100);

    // 输出所有质数
    for (int p : primes) cout << p << " ";  // 2 3 5 7 11 ...
    cout << "\n";

    // 分解质因数（利用 minp 反复除最小质因子）
    auto factor = [&](int x) {
        vector<pair<int, int>> res;  // {质因子, 次数}
        while (x > 1) {
            int p = minp[x], cnt = 0;
            while (x % p == 0) { x /= p; ++cnt; }
            res.emplace_back(p, cnt);
        }
        return res;
    };
    for (auto [p, c] : factor(84)) cout << p << "^" << c << " ";  // 2^2 3^1 7^1
    cout << "\n";
}
```

## 扩展：筛积性函数

利用 `minp` 和 `minp_cnt`（最小质因子的幂次），可以 $O(n)$ 筛出常见积性函数。

```cpp
/**
 * @brief 线性筛 + 积性函数
 *
 * @tparam F  自定义积性函数 f(p, k) = f(p^k)（p 是质数，k ≥ 1）
 * @param n  上界
 * @param f  质数幂处的函数值
 * @return vector<T>  f[0..n]，f[0] 和 f[1] 需自行定义
 *
 * @example
 *   // 筛 φ(n)（欧拉函数）
 *   auto phi = sieve_arithmetic(n, [](int p, int k) {
 *       int pk = 1; for (int i = 0; i < k; ++i) pk *= p;
 *       return pk - pk / p;   // φ(p^k) = p^k - p^{k-1}
 *   });
 */
template <typename T, typename F>
vector<T> sieve_arithmetic(int n, F f) {
    vector<int> primes, minp(n + 1, 0), minp_cnt(n + 1, 0);
    vector<T> res(n + 1, T(0));
    if (n >= 0) res[0] = T(0);   // 按需修改
    if (n >= 1) res[1] = T(1);   // 按需修改

    for (int i = 2; i <= n; ++i) {
        if (!minp[i]) {
            minp[i] = i;
            minp_cnt[i] = 1;
            primes.push_back(i);
            res[i] = f(i, 1);     // f(p^1)
        }
        for (int p : primes) {
            if (i * p > n) break;
            minp[i * p] = p;
            if (i % p == 0) {     // p | i → p 也是 i 的最小质因子
                minp_cnt[i * p] = minp_cnt[i] + 1;
                int pk = 1;
                for (int t = 0; t < minp_cnt[i * p]; ++t) pk *= p;
                // f(i*p) = f(i / p^{cnt}) * f(p^{cnt+1})
                res[i * p] = res[i / pk * p] * f(p, minp_cnt[i * p]);
                break;
            } else {              // p ∤ i
                minp_cnt[i * p] = 1;
                res[i * p] = res[i] * f(p, 1);  // 互质 → 积性
            }
        }
    }
    return res;
}
```

### 常用积性函数示例

```cpp
// 欧拉函数 φ(n)
auto phi = sieve_arithmetic<int>(n, [](int p, int k) {
    int pk = 1; for (int i = 0; i < k; ++i) pk *= p;
    return pk - pk / p;
});

// 莫比乌斯函数 μ(n)
auto mu = sieve_arithmetic<int>(n, [](int p, int k) {
    return k == 1 ? -1 : 0;   // μ(p) = -1, μ(p^k) = 0 (k ≥ 2)
});

// 除数函数 d(n) = 因子个数
auto d = sieve_arithmetic<int>(n, [](int p, int k) {
    return k + 1;             // d(p^k) = k + 1
});

// 约数和 σ(n)
auto sigma = sieve_arithmetic<long long>(n, [](int p, int k) {
    // σ(p^k) = 1 + p + p^2 + ... + p^k
    int pk = 1, sum = 1;
    for (int i = 0; i < k; ++i) { pk *= p; sum += pk; }
    return sum;
});
```

## 复杂度

- **时间：** $O(n)$
- **空间：** $O(n)$（`minp` 数组 + `primes` 列表）

---

## 注意

1. 数组下标从 0 开始，下标 0 和 1 无意义（筛范围是 $[2, n]$）。
2. `minp[i]` 同时可作为**质数判定**：`minp[i] == i` 则 $i$ 为质数。
3. 基于最小质因子分解质因数只需 $O(\log n)$（反复除以 `minp[x]`）。
4. 积性函数筛需要提供 $f(p^k)$ 的计算方式，模板自动利用积性递推。
