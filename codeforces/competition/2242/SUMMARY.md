## C. Unstable Elements

- **题意：** 每次操作标记每个不同值的第一个出现，然后统一删除或复制，问长度为 k 的可达数组数量
- **核心思路：** 
  - 存活集合是频次降序下的前缀，枚举存活个数 s
  - 最终长度 k = sum_s - P * s，判断 (sum-k) % s == 0 且 P < min_f
- **原始代码 bug：** `vector<int> b(a[n-1])` 数组大小少 1 → 改为 `b(a[n-1] + 1)`
- **代码位置：** `C.cpp`

## D. Two Digit Strings

- **题意：** 相邻数字合为和模 10，求两串能变成相等串的最大长度
- **核心思路：**
  - 转化为分段求和模 10，分界点必须 pa[i] == pb[j]
  - DP + 二维前缀最大值优化，滚动数组 O(n*m)
- **原始代码 bug：** 
  - `pb(n+1,0)` 改为 `pb(m+1,0)`
  - 构建 pb 时用了 a[i] 而非 b[i]（笔误）
- **代码位置：** `D.cpp`
